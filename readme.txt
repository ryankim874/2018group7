문서를 변경하실 때 1.4.3 Version Update History에 있는 표에 변경사항을 반드시 추가해주시기 바랍니다. 


1.4.2.	Version Management Policy
본 문서를 수정할 때마다 버전을 업데이트 한다. 
이미 완성한 파트를 변경할 때는 Minor number를 변경하며, 
새로운 부분을 추가하거나 괄목한 만한 변화가 있을 경우 Major number를 변경한다.


새로 입력한 부분의 목차는 [참조]-[목차]-[텍스트 내용 추가]를 해 주신후 [목차 업데이트]-[전체 업데이트]를 해주셔야 추가됩니다.
